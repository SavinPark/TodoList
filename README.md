# ZEROBASE ToDoList Project -- React

## TodoTemplate 컴포넌트
- todolist의 레이아웃을 설정
- 페이지의 중앙에 그림자가 적용된 white box

## TodoHead
- 오늘의 날짜 & 요일 표시
- 아픙로 해야할 일이 몇 개 남았는지 나타냄


## TodoList
- 할 일들에 대한 정보


## TodoItem
- TodoList의 각 항목들
- 왼쪽의 체크아이콘을 누르면 '할 일 완료'
- 오른쪽에 마우스를 올리면 나타나는 휴지통 아이콘을 누르면 '할 일 삭제'


## TodoCreate
- 할 일 등록
- + 를 누르면 할 일을 등록할 수 있는 폼 등장
- X 를 누르면 폼 사라짐
