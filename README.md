# ToDoList Project -- React

## TodoTemplate 컴포넌트
- todolist의 레이아웃을 설정
- 페이지의 중앙에 그림자가 적용된 white box

## TodoHead
- 오늘의 날짜 & 요일 표시
- 남은 할 일의 개수 표시


## TodoList
- 할 일들에 대한 정보
- TodoItem들의 묶음


## TodoItem
- TodoList의 각 항목들
- 각 항목에 마우스 hover시 커서가 포인터 모양으로 바뀌며 회색의 삭제 아이콘이 나타남
- 왼쪽의 체크 아이콘을 누르면 '할 일 완료' 표시 
- 오른쪽에 마우스를 올리면 나타나는 휴지통 아이콘을 누르면 '할 일 삭제'


## TodoCreate
- 할 일 등록
- + 버튼을 누르면 할 일을 등록할 수 있는 등록페이지가 열림
- x 버튼을 누르면 등록페이지가 닫힘
